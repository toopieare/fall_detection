# Fall Detection

## Data folder structure
```bash
├── data                            # Data collected from acceleromer
├── notebook                        # Files containing the fall detection model
│   ├── fall_detection.ipynb        # Jupyter notebook to train a simple fall detection model
│   ├── sample.ino                  # .ino simulator file for testing the fall detection model 
│   ├── requirements.txt            # requirements.txt for training the falling detection model
```
